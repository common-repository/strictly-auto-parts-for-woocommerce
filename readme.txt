== Installation ==

1.   Upload the `sap-shipping` folder to the `/wp-content/plugins/` directory
2.   Activate the plugin through the 'Plugins' menu in WordPress
3.   Visit WooCommerce >> Settings >> Advanced >> REST API
4.   Add Key
5.   After generating the keys, you will get the Consumer Key and Consumer Secret Key. Please copy these keys as it will no longer be visible after closing this screen. Once copied, please share the keys with the Strictly Auto Parts Team.
6.   We will need to add the webhook URLs under the "Webhook" tab. Go to WooCommerce >> Settings >> Advanced >> Webhooks
7.   Add webhook
7.1. Create Order Webhook
7.2. Update Order Webhook
